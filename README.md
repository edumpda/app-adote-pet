# app-adote-pet

Aplicativo desenvolvido para a conclusão da matéria de Desenvolvimento de Aplicativos na Universidade de Brasília.

# Dependecias para rodar o projeto

- npm
- node
- yarn
- git
- expo go (smartphone)

# Rodar o projeto

Dentro do diretoria do projeto

- yarn add expo (se não instalado)
- cd mobile
- npx expo start
- escanear com o expoGo
